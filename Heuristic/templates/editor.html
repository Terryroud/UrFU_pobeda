О INJECTION_PATTERNS

Список паттернов для обнаружения prompt инъекций.

Каждый паттерн на новой строке. Регистр не учитывается. Пример: "ignore", "disregard", "system prompt"

О Vectors

Векторы содержат:

  • name: Уникальное название
  • description: Описание вектора
  • patterns: Список паттернов
  • weight: Вес (0.1-10.0)

Используйте valid JSON формат. Проверяйте синтаксис перед сохранением.

Советы
  • Сохраняйте изменения после редактирования
  • Проверяйте JSON синтаксис для векторов
  • Используйте понятные описания
  • Тестируйте паттерны на реальных примерах
